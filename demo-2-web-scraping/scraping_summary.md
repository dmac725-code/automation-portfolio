# Web Scraping Demo: Pennsylvania Public Libraries

## The Project
Extracted a directory of public libraries across Pennsylvania from publicly available government sources into a clean, structured spreadsheet.

## What I Delivered
A single CSV file with 25 libraries containing:
- Library name
- Full address (street, city, state, zip)
- Phone number (standardized format)
- Website URL
- County

## Process
1. Identified publicly available library directory pages
2. Built a Python scraper to extract structured data from each listing
3. Cleaned and standardized all fields (phone formatting, address normalization)
4. Validated URLs and removed dead links
5. Delivered in CSV format ready for import into any CRM or spreadsheet

## Key Details
- **Data source:** Publicly available government directory (no terms of service violations)
- **Records extracted:** 25
- **Fields per record:** 8
- **Format:** CSV (easily convertible to Excel, JSON, or database import)
- **Quality checks:** All phone numbers verified as valid format, all URLs tested for active status

## Turnaround
Completed in under 3 hours including data validation.
